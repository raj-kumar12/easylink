ProTee Interface Click this link to download ProTee Interface
  • From your downloads folder RIGHT CLICK and select Run as Admisitrator On completion of install, select create a desk top icon and close the installer.

  • The interface will request a licence key.
  • Select Enter Activation Code and a machine ID will be shown.
  • Copy this and send via email to support@protee-united.com with your full name.
  • You will receive in return a licence key which should be copied and pasted into the activation box.
